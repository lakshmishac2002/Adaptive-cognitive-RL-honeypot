# RL-Based Adaptive Honeypot System

## Project Structure
- `data/` - Training data and attack logs
- `models/` - Saved RL models
- `logs/` - Training logs
- `results/` - Evaluation results
- `configs/` - Configuration files
- `scripts/` - Utility scripts

## Quick Start
1. Train agent: `python train_agent.py`
2. Evaluate: `python evaluate_agent.py`
3. Deploy: `python deploy_honeypot.py`
